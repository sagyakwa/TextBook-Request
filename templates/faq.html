I got a link but it has a weird extension like .djvu or .epub and I just want a PDF. What now?

You can try another of the links provided in the application. If that yields no avail, there is always a software to read your file extension. If you simple search google, you'll get some results to play around with. For example, I would search "software to read .epub files". If you really want it in another extension, there are extension converters online. Simple search something like ".epub to .pdf converter". Beware though, it might not come out as pretty as the original.
